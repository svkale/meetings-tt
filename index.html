IITI Mech 3rd Year Time Table

This API (join) takes you to the meeting using the link (if pre-provided by the course instructor) if the lecture is going on, or there are five minutes to start the lecture.

The following features are not present, and are considered in the scope of the project:
  • Take to the meeting scheduled as extra lecture.
  • Cancel an existing lecture.
  • Give user the choice when one lecture is just ending and the other is just starting.
  • Show timetable for the week with extra or cancelled lectures.
  • Show all the links course wise.

The code is published on github. The timetable and the links are taken from the same repository in the form of json files. Any additions / extra feature ideas / creative pull requests are welcome. It is highly recomended to not use this API for exams, as this is can create confusion.

**This is just the project made out of curiosity, and NOT officially provided by the institute.
